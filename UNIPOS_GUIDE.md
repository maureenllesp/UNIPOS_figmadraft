# UNIPOS - Point-of-Sale & Inventory System

A modern, glassmorphic Point-of-Sale and Inventory Management System built with React, TypeScript, and Tailwind CSS.

## Features

### Authentication & Role-Based Access
- Secure login with role-based permissions
- Three distinct user roles with customized interfaces
- Profile management and logout functionality

### Dashboard
- AI-generated insights and predictions
- Sales overview with charts
- Top-selling products
- Low stock alerts
- Peak sales hours analysis
- 7-day sales forecast

### Point-of-Sale (Cashier)
- Clean, modern POS interface
- Product browsing and search
- Shopping cart with quantity management
- Real-time order totals
- Quick checkout process

### Products Management
- Product catalog with images
- Stock level monitoring
- Add, edit, delete products (Admin)
- Category organization
- Pricing and inventory tracking

### Transactions
- Complete transaction history
- Searchable and filterable records
- Payment method tracking
- Transaction status monitoring
- Export to CSV functionality

### Inventory Management
- Real-time stock levels
- Low stock warnings
- Inventory trend analysis
- Stock update capabilities (Admin)
- Supplier tracking
- View-only access for Business Owners

### Sales Analytics
- Daily, weekly, and monthly reports
- Revenue and profit tracking
- Sales by category breakdown
- Payment method analytics
- Interactive charts and visualizations

### Refund Management
- Refund request processing
- Approve/reject workflows
- Refund history tracking
- Reason documentation

### AI Insights (Business Owner)
- Sales predictions and forecasts
- Inventory optimization recommendations
- Peak hour staffing suggestions
- Product bundle opportunities
- Customer behavior analysis
- 30-day sales forecast with confidence intervals

### Settings
- Store information management
- Tax and currency configuration
- Receipt layout customization
- Notification preferences
- Security settings
- Password management

### User Management (Admin)
- Create and manage user accounts
- Role assignment
- User activation/deactivation
- Password reset
- Activity tracking

## Design Features

- **Glassmorphic UI:** Modern glass-like design with blur effects
- **Color Palette:** Soft blue (#5b8db8), beige/cream (#f5f1e8), muted grays
- **Responsive:** Optimized for desktop and tablet views
- **Professional Typography:** Clean Inter font family
- **Rounded Cards:** Consistent 1rem border radius
- **Charts & Visualizations:** Powered by Recharts
- **Smooth Transitions:** Professional animations and hover effects

## Technology Stack

- React 18
- TypeScript
- Tailwind CSS v4
- Radix UI Components
- Recharts for data visualization
- Lucide React for icons

## Quick Start

1. Click on the login page

3. Explore the system based on your role
